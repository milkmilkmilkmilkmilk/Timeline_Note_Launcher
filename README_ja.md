# Timeline Note Launcher

Obsidian用のTwitter風タイムラインビューで、ノートをランダムにレビューできます。ノートカードを縦リストで表示し、タップで開くナビゲーションを提供します。

## 機能

- **タイムラインビュー**: Twitter風の縦型フィードでノートをカード表示
- **複数の選択モード**:
  - ランダム: 純粋なランダム選択
  - 古さ優先: 古いノートがより頻繁に表示
  - SRS（間隔反復システム）: SM-2アルゴリズムによる最適なレビュー間隔
- **SRS対応**: Again/Hard/Good/Easyボタンでカードを評価、次回レビューを自動スケジュール。取り消しボタンで評価を元に戻せます
- **引用ノート**: 現在のノートから引用テキスト付きの新規ノートを作成
- **リンクノート**: ノート間にリンクを追加
- **コメント機能**: タイムスタンプ付きのCalloutコメントをノートに直接追加
- **クイックノート**: タイムラインから直接新規ノートを作成
- **フィルターバー**: テキスト検索、ファイルタイプ・タグ・日付範囲でフィルタリング。プリセットの保存・読み込み対応
- **グリッド/リスト切替**: カードレイアウトを切り替え
- **マルチファイル対応**: Markdown、テキスト、画像、PDF、音声、動画、Excalidraw、Canvas、Jupyter Notebook、Officeファイル
- **ブックマーク連携**: コアブックマークプラグインと連携してカードのブックマークを切り替え
- **YAML連携**: フロントマターから難易度・優先度・日付を読み込み
- **統計ダッシュボード**: アクティビティヒートマップ、レビュー統計、ファイルタイプ別集計
- **キーボードショートカット**: ホットキーで効率的にナビゲート
- **カラーテーマ**: 複数のアクセントカラーとUIテーマ（クラシック / Twitter風）
- **無限スクロール**: スクロールで追加カードを読み込み
- **プルトゥリフレッシュ**: モバイルで下にスワイプして更新
- **分割ビュー**: デスクトップで分割ペインにノートを表示
- **フロントマタープロパティ**: 選択したフロントマタープロパティをカードに表示

## インストール

### コミュニティプラグインから（推奨）

1. Obsidianの設定を開く
2. コミュニティプラグインに移動し、セーフモードを無効化
3. 閲覧をクリックし、「Timeline Note Launcher」を検索
4. インストールして有効化

### 手動インストール

1. 最新リリースから `main.js`、`manifest.json`、`styles.css` をダウンロード
2. Vaultの `.obsidian/plugins/` ディレクトリに `timeline-note-launcher` フォルダを作成
3. ダウンロードしたファイルをフォルダにコピー
4. Obsidianを再読み込みし、設定 > コミュニティプラグインでプラグインを有効化

## 使い方

### タイムラインを開く

- 左リボンのロケットアイコンをクリック
- またはコマンドパレットから: 「Timeline Note Launcher: Open Timeline」

### ノートをレビューする

1. 選択したモードに基づいてカードがタイムラインに表示
2. カードをクリックしてノートを開く
3. SRSモードでは、難易度ボタンで記憶度を評価:
   - **Again**: 10分後に再レビュー
   - **Hard**: 短い間隔
   - **Good**: 通常の間隔
   - **Easy**: ボーナス付きの長い間隔

### 引用ノートを作成する

1. タイムラインでカードを開く
2. 引用したいテキストを選択
3. 「Add Quote」をクリックして選択を収集
4. 「Create Quote Note」をクリックして引用付きの新規ノートを生成

### コメントを追加する

1. カードのコメントボタンをクリック
2. コメントを入力
3. タイムスタンプ付きのCalloutとして元のノートに追記

## 設定

| 設定項目 | 説明 |
|---------|------|
| Target Folders | タイムラインに含めるフォルダ（空 = すべて） |
| Exclude Folders | タイムラインから除外するフォルダ |
| Target Tags | タグでノートをフィルタリング |
| Selection Mode | ランダム / 古さ優先 / SRS |
| View Mode | リストまたはグリッドレイアウト |
| Media Size | 画像・埋め込みの最大高さ |
| Preview Mode | 固定行数 / 半分 / 全文プレビュー |
| Color Theme | タイムラインのアクセントカラー |
| UI Theme | クラシックまたはTwitter風レイアウト |
| Show Properties | カードにフロントマタープロパティを表示 |
| Max Cards | タイムラインの最大カード数 |
| Infinite Scroll | スクロールで追加カードを読み込み |
| Auto Refresh | タイムライン自動更新の間隔 |
| YAML Keys | フロントマターから難易度・優先度・日付を読み込み |
| Quick Note Folder | クイックノートの保存フォルダ |
| New Cards per Day | SRSモードでの1日あたりの新規カード上限 |
| Review Cards per Day | SRSモードでの1日あたりのレビュー上限 |
| Initial Interval | 初回レビュー間隔（日数） |
| Easy Bonus | Easy評価時の倍率 |

全設定はプラグイン設定タブで確認できます。

## キーボードショートカット

| キー | アクション |
|------|----------|
| `j` / `ArrowDown` | 次のカード |
| `k` / `ArrowUp` | 前のカード |
| `Enter` / `o` | 選択したノートを開く |
| `1` - `4` | カードを評価（Again/Hard/Good/Easy） |
| `u` | 最後の評価を取り消し |
| `b` | ブックマークを切り替え |
| `c` | コメントモーダルを開く |
| `q` | 引用ノートモーダルを開く |
| `l` | リンクノートモーダルを開く |
| `r` | タイムラインを更新 |
| `Escape` | フォーカスを解除 |

## 動作環境

- Obsidian v1.0.0 以降

## 既知の制限事項

- モバイルではPDFプレビューは非対応（タイムラインにはOpenボタン付きフォールバックを表示）
- 大規模なVaultでは初回読み込みに時間がかかる場合があります
